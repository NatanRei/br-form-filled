Política de Privacidade — BR Form Filler

Última atualização: junho de 2026

Resumo

O BR Form Filler não coleta, armazena em servidor, vende ou compartilha nenhum dado pessoal. Tudo o que a extensão faz acontece localmente, dentro do seu navegador.

O que a extensão lê

Quando você clica no ícone da extensão, ela lê os campos de formulário (input, select, textarea e componentes de dropdown) presentes na aba ativa naquele momento, com o único objetivo de detectar o tipo de cada campo e preenchê-lo com dados de teste gerados localmente. Essa leitura só acontece quando você clica no ícone — a extensão não roda em segundo plano nem observa páginas sem você interagir com ela.

O que a extensão armazena

A extensão guarda, usando a API chrome.storage.local do próprio navegador:

  • Uma identificação (fingerprint) do conjunto de campos de cada formulário já visto, pra lembrar qual tipo de dado você escolheu pra cada campo;
  • O tipo escolhido pra cada campo (ex: "CPF", "E-mail", "Cidade").

Esses dados ficam só no seu navegador, no seu próprio computador. A extensão não tem servidor, não faz nenhuma chamada de rede, e essa informação nunca sai do seu navegador.

O que a extensão NÃO faz

  • Não envia nenhum dado pra servidor algum;
  • Não coleta CPF, e-mail, nome ou qualquer outro dado real que você tenha digitado — só gera dados fictícios próprios;
  • Não usa cookies, analytics ou rastreamento de qualquer tipo;
  • Não vende nem compartilha informação com terceiros, porque simplesmente não existe informação saindo do seu navegador pra compartilhar.

Permissões usadas

  • activeTab — pra ler e preencher o formulário só na aba ativa, só quando você clica no ícone;
  • scripting — pra injetar o script de leitura/preenchimento sob demanda (não fica residente em toda página);
  • storage — pra guardar a configuração localmente, como descrito acima.

Código aberto

Todo o código está disponível publicamente em github.com/NatanRei/br-form-filled — qualquer pessoa pode verificar exatamente o que a extensão faz.
